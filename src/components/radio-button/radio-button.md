```jsx
<RadioButton />
```

With a label:

```jsx
<RadioButton label="Public access" />
```

With a label and description:

```jsx
<RadioButton
  label="Public access"
  description="The repository would be available to anyone"
/>
```

Disabled state:

```jsx
<RadioButton
  label="Public access"
  description="The repository would be available to anyone"
  disabled={true}
/>
```
